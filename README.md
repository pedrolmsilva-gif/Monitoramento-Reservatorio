## Funcionalidades
- Monitoramento de 5 níveis de reservatório, do mais baixo ao mais alto.
- Utilização da biblioteca `colorama` para diferenciar estados por cores:
  - 🔴 Vermelho: Muito baixo (crítico)
  - 🟡 Amarelo: Baixo
  - 🟢 Verde: Médio
  - 🔵 Ciano: Alto
  - 🔵 Azul: Muito alto (alerta)
- Estrutura baseada em lista e dicionários para fácil manutenção.

## Como executar
1. Certifique-se de ter o Python instalado.
2. Instale a biblioteca necessária via terminal:
   ```bash
   pip install colorama
